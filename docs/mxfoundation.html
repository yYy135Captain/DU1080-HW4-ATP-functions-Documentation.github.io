MxFoundation

Overview

This module is designed to drive ARINC429 and ARINC708 functions in MaxT FM429 Device.

# MxFoundation Functions

MxFoundationInitialize

Header

int DLLEXPORT __stdcall MxFoundationInitialize(void);

Purpose

Initializes the MxFoundation system, obtains the device handle, configures the system time base, and initializes the ARINC 429 and ARINC 708 Channels.

Parameter

This function has no parameters.

Return Value

Error Description
0 Initialization completed successfully.
Non-zero An MxFoundation, device, ARINC 429, OR ARINC 708 initialization operation failed.

Example

int32 error = 0; error = ARINC429ChannelInitialize();

MxFoundationClose

Header

void DLLEXPORT __stdcall MxFoundationClose(void);

Purpose

Terminates the MxFoundation system connection disconnects from the server, and resets the initialization status.

Parameter

This function has no parameters.

Return Value

This function does not return a value.

Example

MxFoundationClose();

# ARINC429 Functions

ARINC429ChannelInitialize

Header

int DLLEXPORT __stdcall ARINC429ChannelInitialize(void);

Purpose

Initializes all ARINC 429 transmit and receive channels on the MxFoundation device. The function obtains the module handles, configures module attributes, retrieves channel handles, and prepares the channels for subsequent ARINC 429 transmit and receive operations.

Parameter

This function has no parameter.

Return Value

Error Description
0 All ARINC 429 channels were initialized successfully.
Non-zero One or more ARINC 429 initialization operations failed.

Arinc429SetupReceiveData

Header

int DLLEXPORT __stdcall Arinc429SetupReceiveData(uint64 receive_channel, BOOL speed, int parity, uint64 count, HMXF_BUFFER *receive_buffer);

Purpose

Configures an ARINC 429 receive channel by setting the receive speed and parity, allocating the receive buffer, and starting the receive acquisition process.

Parameter

Parameter Type Description
receive_channel uint64 Receive channel index to configure.
speed BOOL Receive speed selection (0 = 12.5 kbps, non-zero = 100 kbps).
parity int Parity mode (0 = even, 1 = odd).
count uint64 Number of ARINC 429 data records to allocate.
*receive_buffer HMXF_BUFFER Pointer that receives the allocated receive buffer handle.

Return Value

Error Description
0 Receive channel configured successfully.
Non-Zero Failed to configure the receive channel or allocate/start the receive buffer.

Arinc429SendData

Header

int DLLEXPORT __stdcall Arinc429SendData(HMXF_BUFFER *arinc429_transmit_buffer, HMXF_SCHED *arinc429_schedule_handle, uint64 transmit_channel, int send_rate, BOOL speed, int parity, unsigned int transmit_data, unsigned int delay);

Purpose

Configures an ARINC 429 transmit channel and sends data using a periodic transmission schedule.

Parameter

Parameter Type Description
*arinc429_transmit_buffer HMXF_BUFFER Stores the transmit buffer handle.
*arinc429_schedule_handle HMXF_SCHED Stores the periodic schedule handle.
*transmit_channel uint64 Selects the ARINC 429 transmit channel.
send_rate int Sets the periodic transmission rate.
speed BOOL Selects transmit speed.
parity int Selects Parity.
transmit_data unsigned int Data word to transmit.
delay unsigned int Delay applied after starting transmission.

Return Value

Error Description
0 Data transmission setup completed successfully.
Non-Zero An error occurred while configuring or starting transmission.

Arinc429ReceiveData

Header

int DLLEXPORT __stdcall Arinc429ReceiveData(uint64 count, HMXF_BUFFER receive_buffer, uint32 *received_data);

Purpose

Reads ARINC 429 records from a receive a acquisition buffer and returns the data word from the first received record.

Parameter

Parameter Type Description
count uint64 Number of ARINC 429 records for which host-buffer space is allocated.
receive_buffer HMXF_BUFFER Handle of the receive acquisition buffer to read.
*received_data uint32 Pointer that receives the data word from the first ARINC 429 record.

Return Value

Error Description
0 The receive-buffer read completed successfully.
Non-Zero Memory allocation or an ARINC 429 receive operation failed.

Example 

uint32 received_data = 0;
HMXF_BUFFER receive_buffer = 0;
arinc429_error = Arinc429SetupReceiveData(receive_channel, 0, 0, 1, &receive_buffer);

Arinc429FreeBuffers

Header

void DLLEXPORT __stdcall Arinc429FreeBuffers(HMXF_BUFFER arinc429_transmit_buffer, HMXF_BUFFER arinc429_receive_buffer, HMXF_SCHED arinc429_schedule_handle);

Purpose

Releases the ARINC 429 periodic transmission schedule and transmit buffer. It also stops and closes the receive acquisition buffer.

Parameter

Parameter Type Description
arinc429_transmit_buffer HMXF_BUFFER Handle of the periodic transmit buffer to release.
arinc429_receive_buffer HMXF_BUFFER Handle of the receive acquisition buffer to stop and close.
arinc429_schedule_handle HMXF_SCHED Handle of the periodic transmission schedule to release.

Return Value

This function does not return a value.

Example 

Arinc429FreeBuffers(transmit_buffer, receive_buffer, schedule_buffer);

Arinc429SelfTest

Header

int DLLEXPORT __stdcall Arinc429SelfTest(uint64 transmit_channel, uint64 receive_channel, unsigned int data);

Purpose

Performs an ARINC 429 loopback self-test by transmitting a data word, receiving it on the specified channel, and releasing the allocated communication resources.

Parameter

Parameter Type Description
transmit_channel uint64 ARINC 429 transmit channel used for the self-test.
receive_channel uint64 ARINC 429 receive channel used for the self-test.
data unsigned int ARINC 429 data word transmitted during the self-test.

Return Value

Error Description
0 The ARINC 429 self-test completed successfully.
Non-Zero One or more ARINC 429 data word transmitted during the self-test. 
# ARINC708 Functions

ARINC708ChannelInitialize

Header

int DLLEXPORT __stdcall ARINC708ChannelInitialize(void);

Purpose

Initializes the ARINC 708 modules and retrieves the transmit and receive handles required for ARINC 708 communication.

Parameter

This function has no parameters.

Return Value

Error Description
0 ARINC 708 modules and channels were initialized successfully.
Non-Zero An ARINC 708 module or channel initialization operation failed.

Example 

error = ARINC708ChannelInitialize();

Arinc708SetupReceiveData

Header

int DLLEXPORT __stdcall Arinc708SetupReceiveData(uint64 receive_channel, HMXF_BUFFER *receive_buffer);

Purpose

Configures and ARINC 708 receive channel by allocating a receive acquisition buffer and starting the receive acquisition process.

Parameter

Parameter Type Description
receive_channel uint64 Index of the ARINC 708 receive channel to configure.
receive_buffer HMXF_BUFFER * Pointer that receives the allocated receive buffer handle.

Return Value

Error Description
0 The receive channel was configured successfully.
Non-Zero Failed to allocate or start the receive acquisition buffer.

Example 

HMXF_BUFFER  receive_buffer = 0;
arinc708_error = Arinc708SetupReceiveData(receive_channel, &receive_buffer);

Arinc708SendData

Header

int DLLEXPORT __stdcall Arinc708SendData(HMXF_BUFFER *arinc708_transmit_buffer, HMXF_SCHED *arinc708_schedule_handle, uint64 transmit_channel, int send_rate, unsigned int transmit_data);

Purpose

Configures an ARINC 708 transmit channel, creates a periodic transmission schedule, prepares the transmit records, and states periodic data transmission.

Parameter

Parameter Type Description
arinc708_transmit_buffer HMXF_BUFFER * Pointer that receive the transmit buffer handle.
arinc708_schedule_handle HMXF_SCHED * Pointer the receives the periodic schedule handle.
transmit_channel uint64 Index of the ARINC 708 transmit channel.
send_rate int Periodic transmission rate.
transmit_data unsigned int Data value written into the transmit records.

Return Value

Error Description
0 Periodic transmission was configured successfully.
Non-Zero Failed to configure or start the ARINC 708 transmission.

Arinc708ReceiveData

Header

int DLLEXPORT __stdcall Arinc708ReceiveData(uint64 count, HMXF_BUFFER receive_buffer, uint32 *received_data);

Purpose

Reads ARINC 708 records from a receive acquisition buffer and returns a data value from the first received record.

Parameter

Parameter Type Description
count uint64 Number of ARINC 708 records for which host-buffer space is allocated.
receive_buffer HMXF_BUFFER Handle of the receive acquisition buffer to read.
*received_data uint32 Pointer that receives the selected data value from the first ARINC 708 record.

Return Value

Error Description
0 The receive-buffer read completed successfully.
Non-Zero Memory allocation or an ARINC 708 receive operation failed.

# MISC Helper

SetParityBit

Header

int DLLEXPORT SetParityBit(unsigned int arinc_word, unsigned int *result, int parity);

Purpose

Calculates and sets the Most Significant Bit (MSB) parity bit for a 32-bit ARINC 429 word based on the selected parity mode.

Parameter

Parameter Type Description
arinc_word unsigned int The 32-bit input word (with or without a pre-existing parity bit).
*result unsigned int The expected ARINC word with parity.
parity int The parity type selection (ARINC_PARITY_EVEN, ODD, or NONE).

Return Value

Error Description
0 The parity bit was calculated successfully.
