1.43 InitializeUART

1.43.1 Header 

int DLLEXPORT __stdcall InitializeUART(ViSession *session_handle, const char *resource_handle, ViBoolean *test_result, char *status_message);

1.43.2 Purpose

Initializes SerialUART communication between two serial ports that transmit data (TX) and receive data (RX). Handles RS-232 and RS-422 serial protocols.

1.43.3 Parameters

Parameter Type Purpose
session_handle ViSession * session handle
resource_handle const char * resource handle
test_result ViBoolean * test result
status_message char * status message of VISA port and manager

1.43.4 Return Value

Error Code Error
0 Success
-1 VISA cannot open resource handle manager
-2 VISA cannot open port

1.44 UARTSetting

1.44.1 Header 

int DLLEXPORT __stdcall UARTSetting(ViSession *session_handle, ViUInt32 baud_rate, char *status_message)

1.44.2 Purpose

This function configures the port settings of a serial port given a session handle and baud rate.

1.44.3 Parameters

Parameter Type Purpose
session_handle ViSession * unique session handle
baud_rate ViUInt32 baud rate transmission
status_message char * status of serial port configuration

1.44.4 Return Value

Error Error Code
0 Success
-1 Serial port setting failed

1.46 ReplyUART

1.46.1 Header 

int DLLEXPORT __stdcall ReplyUART(ViSession *session_handle, char *status_message, int time_out, int number_bytes)

1.46.2 Purpose

Read/write data until the amount is equal to number_bytes

1.46.3 Parameters

Parameter Type Purpose
session_handle ViSession * unique session handle
status_message char * status of VISA
time_out int amount of time (in seconds) the program waits for a return value before breaking out
number_bytes int total number of bytes to read

1.46.4 Return Value

Error Error Code
0 Success
-1 VISA write failed

1.48 SerialVISAInit

1.48.1 Header 

int DLLEXPORT __stdcall SerialVISAInit(const char *port_name, int baud_rate, int data_bits, int parity, int stop_bits, int flow_control, int time_out)

1.48.2 Purpose

RS-232 and RS-422 send and read verification based on user configured serial settings.

1.48.3 Parameters

Parameter Type Purpose
port_name char * port name
baud_rate int rate of byte transmission
data_bits int data bits
parity int parity
stop_bits int stop bits
flow_control int flow control
time_out int time out in seconds

1.48.4 Return Value

Error Error Code
0 Success
-1 Could not open VISA resource_handle manager
-2 Could not open VISA port
