1.9 InitializeUART

1.9.1 Header 

int DLLEXPORT __stdcall InitializeUART(ViSession *session_handle, const char *resource_handle, ViBoolean *test_result, char *status_message);

1.9.2 Purpose

Initializes SerialUART communication between two serial ports that transmit data (TX) and receive data (RX). Handles RS-232 and RS-422 serial protocols.

1.9.3 Parameters

1.9.4 Return Value

Error Error code
0 no errors and initialization was a success
-1 VISA cannot open resource handle manager
-2 VISA cannot open port associated with resource handle
